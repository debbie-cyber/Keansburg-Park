# KEANSBURG PARK

A single-page marketing website for an amusement park, designed to provide a comprehensive look at rides, tickets, and park information in one view. This project is a strong demonstration of responsive design principles and foundational DOM manipulation using JavaScript.

## Key Features

* ingle-Page Layout (SPA concept): All content is accessible without page reloads.
* Interactive Elements: Utilizes JavaScript for smooth navigation.
* Modern CSS: Includes complex layouts (Flexbox/Grid) for an engaging user experience.

---

## Getting Started

This project is entirely front-end and requires no server or compilation. The easiest and most professional way to view it is by using the **VS Code Live Server extension**.

### Option 1: Running with VS Code Live Server (Recommended)

This method automatically launches the site in your browser and instantly refreshes as you edit the code.

#### Prerequisites
1.  **VS Code:** Installed on your machine.
2.  **Live Server Extension:** Install this extension by Ritwick Dey from the VS Code Marketplace.

#### Steps
1.  Open the project folder in VS Code.
2.  Right-click on your main HTML file (i.e. Kansburg Park.html).
3.  Select **"Open with Live Server."** The site will launch in your default web browser.

---

### Option 2: Direct Browser Launch

You can view the project without any extensions by directly opening the file.

1.  Navigate to your project folder on your computer.
2.  Double-click on the main HTML file (i. e. Keansburg Park.html).
3.  The site will open in your default web browser.
